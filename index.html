TOSS-Codec: A Temporal-Offset-Supervised Streaming Audio Codec for Mobile Devices

TOSS-Codec is a compact family of fully causal streaming neural audio codecs for device-oriented audio compression and transmission. Temporal Offset Supervision supervises a delayed causal reconstruction with an earlier ground-truth target, turning a small explicit output delay into a controllable quality-latency trade-off while keeping the inference architecture unchanged.

Overview

Streaming architecture and Temporal Offset Supervision at a glance.

This figure summarizes the strictly causal encoder-RVQ-decoder pipeline, shifted waveform-loss alignment, and causal streaming inference with explicit output delay and tail padding.

Method

A causal codec trained to use output delay as supervision.

The proposed codec is designed for endpoint audio communication: online reconstruction, flexible bitrate adaptation, and compact CPU-friendly inference. TOSS improves the same strictly causal computation graph by changing only the training-time temporal alignment, rather than adding non-causal layers, lookahead buffers, or extra parameters.

01

Causal backbone

The encoder compresses the waveform into a low-frame-rate latent sequence; the decoder reconstructs online using causal blocks only.

02

Residual quantization

RVQ transmits code indices instead of continuous latents, and the number of active quantizers controls bitrate.

03

Temporal Offset Supervision

TOSS supervises delayed reconstruction x_hat[t + D_T] with the earlier target x[t].

04

Auditable streaming delay

At inference, the model emits samples with an explicit delay of D_T / fs; the computation graph remains causal.

Codec pipeline

Causal encoding, scalable RVQ, delayed supervision.

The transmitted bitstream contains only quantizer indices. Bitrate is selected by choosing the number of active residual quantizers at inference time, and each active quantizer contributes 0.625 kbps at 16 kHz.

Training stability

Validation Mel loss under different temporal offsets.

All validation curves decrease smoothly without divergence, indicating that TOSS can be optimized with the same adversarial codec training pipeline as standard sample-aligned supervision.
